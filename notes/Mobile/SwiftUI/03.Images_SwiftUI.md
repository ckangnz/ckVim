# Images

## Importing the image

1. Drag into Asset Catalog's editor

- This will create a new image set for the image.

```
import SwiftUI

struct CircleImage: View {
  var body: some View {
    Image("imageName")
      .clipShape(Circle())
      .overlay {
        Circle().stroke(.gray, lineWidth: 4)
      }
  }
}

#Preview {
  CircleImage()
}
```
